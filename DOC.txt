Перед первым запуском приложения обязательно к прочтению.
Системные требования:
	Операционная система ubuntu

Для запуска приложения необходимо:
	1). В терменале ввести git clone https://github.com/Kuplays/TODO-LIST-course-project-trpo.git
	2). Прописать cd TODO-LIST-course-project-trpo.git
	3). Ввести команду make
	4). Ввести cd bin/
	5). Ввести ./todo.exe

Поздравляем приложение запущено!

Работа с приложением:
	В меню приложения 5 пунктов ([1] NEW ENTRY - Создание и ввод новой задачи; [2] SHOW ALL ENTRIES - Вывод введенных данных в порядке ввода пользователем; [3] SHOW ALL ENTRIES BY DATE - Вывод введенных данных, данные отсортированы по дате; [4] SHOW IMPORTANT ENTRIES - Вывод введенных данных, которые имеют пометку IMPORTANT; [5] TERMINATE - Выход из приложения)

	Для того чтобы выбрать один из предложенных вариантов работы программы нужно ввести число (1-5). неподходящие цифры, сочетания символов и букв программой не воспринимаются.

	>[1] NEW ENTRY. (ENTER TITLE) - в данном поле пользователь вводит заголовок/название запланированного дела. Ввод происходит на английском языке. (ENTER DESCRIPTION) - в данном поле пользователь описывает запланированное дело, так же поле может остаться пустым, для этого необходимо нажать клавишу ENTER. (Is it important? [y/n]) - в данном поле пользователь указывает важное дело или нет. Необходимо ввести с клавиатуры "y" - важное или "n" - не важное. Другие символы, числа и сочетания клавиш запрещены для ввода.

	>[2] SHOW ALL ENTRIES. Программа выводит введенные данные в порядке ввода пользователем; Важны дела выделены красным цветом. Для того, чтобы вернуться в главное меню необходимо нажать клавишу "ENTER".

	>[3] SHOW ALL ENTRIES BY DATE. Программа выводит введенные данные пользователем; выведеные данные на экран будут отсортированы по дате; Важны дела выделены красным цветом. Для того, чтобы вернуться в главное меню необходимо нажать клавишу "ENTER".

	>[4] SHOW IMPORTANT ENTRIES. Программа выводит только те введенные данные пользователем, которые имеют пометку IMPORTANT("y"); Данные с пометкой "n" выведены не будут. Для того, чтобы вернуться в главное меню необходимо нажать клавишу "ENTER".

	>[5] TERMINATE. Выход из приложения осуществляется вводом цифры "5".

Удачного пользования!
